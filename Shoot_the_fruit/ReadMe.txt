To run the game:

1; open command prompt
2; type: pgzrun (and one space)
3; drag the file from file explorer to the command prompt window

Should get something similar: 
pgzrun "C:\Users\Laszlo\Desktop\PythonGames\Shoot_the_fruit\shoot.py"


if while dropping, the 'quote marks' were not placed, navigate to the begining
		   and the end of the address and place "" them.
